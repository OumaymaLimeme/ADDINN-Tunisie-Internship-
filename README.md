# ADDINN-Tunisie-Internship-

During my internship at ADDINN, the proposed solution aimed to automate its recruitment process through the development of the 'e-recruitment' platform. This platform significantly streamlined the pre-selection of candidates, optimizing the hiring workflow. To enhance security and facilitate secure user authentication and authorization, the platform incorporated JSON Web Tokens (JWT). The use of JWT technology enabled secure token generation and exchange, boosting the platformâ€™s integrity and confidentiality. This automation not only sped up the recruitment process but also ensured that sensitive candidate data and interactions were protected with strong security protocols.
